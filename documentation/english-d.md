# rhino-bytearray

## 1. Get Started
```javascript
const ByteArray = require('{path}/index.js').ByteArray;
```
If function 'require' is not defined in your code, see <a href="https://github.com/NoBrain0917/Rhino-in-require">here</a>.

## 2. Use
### ByteArray(size: Number)
```javascript
let b = ByteArray(size);
```
